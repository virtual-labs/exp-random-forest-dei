### This folder contains all the JS files used in the simulation.

| File | Description |
|------|-------------|
| `main.js` | Main simulation logic and step management |
| `main_new.js` | Updated/alternate simulation logic |
| `rf_animation.js` | Random Forest animation logic |
| `rf_animator.js` | Random Forest animator component logic |
| `rf_data_binary.js` | Random Forest data for binary classification |
| `rf_data_multiclass.js` | Random Forest data for multi-class classification |
| `tree_data_binary.js` | Decision tree data for binary classification |
| `tree_data_multiclass.js` | Decision tree data for multi-class classification |
